# 虾听音乐

vue3实战项目，服务接口感谢[NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi)支持提供

前端技术栈:

vue3、vuex、vueRouter、vite、vant-ui、typeScript

单位使用 Viewport 自适应
